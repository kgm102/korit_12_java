# Java 기초
## 정의
- 객체 지향 프로그래밍(OOP : object-oriented Programming)언어,
Write Once, Run Anywhere라는 철학을 바탕으로 한 번 작성했을 때 다양한 OS에서 작동될 수 있도록 함. 기본적으로 자가
-  가상 머신(JVM : Java virtual Machine)을 통해 프로그램이 실행되며, 이로 인해
-  OS가 하드웨어 차이에 상과넚이 길관된 실행환경을 제공.

## Java 언어의 개발 배경
- 호환성과 이식성을 극대화 하기 위해 개발되. 그래서 Java 관련 시험 문제 중에
-  '플랫폼 독립성'이라는 표현이 자주 등장합니다.(정처기/정처산기 등). 운영체제에 관계없이
- 모든 플랫폼에서 실행될 수 있다느느 점에서 큰 이점. 이는 Java가 '컴파일'되어 바이트
- 코드로 변환된 이후에 JVM에서 실행되기 때문.
